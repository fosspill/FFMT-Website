# Managing your mods

## Enabling and Disabling

Edit the modlist.cfg file within your operatingsystem's configuration directory. 
Set Enabled to True or False depending on what you want.


 Then run:
 ```
 > ffmt mr
 ```
 or
 ```
 > ffmt mods refresh
 ```

 
## Disable all mods

 ```
 > ffmt mods disable
 ```

## Enable all mods

 ```
 > ffmt mods enable
 ```
